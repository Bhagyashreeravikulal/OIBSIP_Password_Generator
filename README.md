This project is a **Random Password Generator** developed using **Python and Tkinter**.
The application allows users to generate secure and random passwords based on selected options such as letters, numbers, and symbols.
It provides a simple **Graphical User Interface (GUI)** where users can choose password length and character types.

This project was created as a **BCA Mini Project** to demonstrate the use of Python programming and GUI development.

---

## Features

* Generate random secure passwords
* Select password length
* Include letters, numbers, and symbols
* Copy password to clipboard
* Simple and user-friendly GUI
* Error handling for invalid inputs

---

## Technologies Used

* Python
* Tkinter (GUI library)
* Random module
* String module

## Installation

1. Install **Python (version 3.x)** on your system.
2. Download or clone this repository.

```
git clone https://github.com/yourusername/password-generator-python.git
```

3. Navigate to the project folder.

```
cd password-generator-python
```

---

## How to Run

Run the following command in terminal or command prompt:

```
python password_generator.py
```

A GUI window will open where you can generate passwords.

---

## Example Usage

1. Enter the desired password length.
2. Select the character types (letters, numbers, symbols).
3. Click **Generate Password**.
4. Copy the password if needed.
